# amy_daniel_python_challenges

Python Challenges for AUG 25 CODE:You DA

Here are the types of challenges coevered in this repo:

Reverse a String – Work with indexing and loops
Check for a Palindrome – Combine string methods, logic, and functions
Find the Maximum Number in a List – Use iteration and/or built-in tools
Check if a Number is Prime – Build a logical test using conditionals and loops
Caesar Cipher Encoder/Decoder – Practice string manipulation and modular arithmetic
You’ll also have the option to take it further with challenges like:

Count Word Frequency in a Text File – Practice file reading and dictionaries
Write and Save a Simple Log File – Combine user input and file writing
Build a Basic Calculator – Create a menu-driven function with multiple operations
Password Strength Checker – Use string methods, loops, and logic
FizzBuzz with a Twist – Reinforce control flow and add variation to a familiar task
